# Internationalization (i18n)

Translations are defined in `I18N.cs` and stored in the static `I18N.set` object. Each language has an `Entries` dictionary mapping keys to localized strings.

### Key Format

- Use `YourAppNamespace.<ActionOrItemName>` as the key.
- For action names, match the method name passed to `AddAction`.

Example English entries:

```csharp
I18N.set.Translations["en-US"].Entries["YourApp.ShowChannels"] = "Channels";
```

Example Polish entries:

```csharp
I18N.set.Translations["pl-PL"].Entries["YourApp.ShowChannels"] = "Kanały";
```

### Runtime Access

The application constructs a `StaticTranslator` in its constructor:

```csharp
public YourApp() : base(new StaticTranslator(I18N.set)) { }
```

Controls and actions retrieve translations via:

```csharp
// simple key for UI elements
var label = Application?.Translator?["YourApp.ShowSettings"];

// for actions the full type name is often used, e.g. Media.MediaApp.GetSettings
var actionLabel = Application?.Translator?["Media.MediaApp.GetSettings"];
```

Remember to rebuild the project after editing `I18N.cs` to apply new translations.

Common translation keys used in the UI include:

```csharp
// page headers
"UI.ChannelList.Header"  // "Channels"
"UI.Playlist.Title"      // "Playlist"
"UI.Settings.Header"     // "Settings"

// buttons and actions
"UI.Refresh.All"         // "Refresh all playlists"
"UI.Save"                // "Save"
"UI.Cancel"              // "Cancel"
"UI.Add"                 // "Add"
"UI.Playlist.Add"        // "+ Add playlist"

// input helpers
"UI.Aria.ChannelName"          // "Channel name"
"UI.Placeholder.PlaylistName"  // "Playlist name"
"UI.Aria.PlaylistName"         // "Playlist name"
"UI.Placeholder.PlaylistUrl"   // "Playlist URL"
"UI.Aria.PlaylistUrl"          // "Playlist URL"
```

You can extend this list with any additional keys your components require.  Actions may use the full namespace + class + method name, for example:

```csharp
I18N.set.Translations["en-US"].Entries["Media.MediaApp.GetSettings"] = "Settings";
```
