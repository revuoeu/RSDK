# RSDK Howto

This folder is the consolidated SDK guidance for RSDK.

**Primary sources:** `RSDK.Client\Howto`, `DalleR\Howto`, `AppTemplate\Howto`, `NewProject\Howto`, and selected generic notes from `Chat\Docs\HowTo`.

## Contents

| File | Purpose |
|------|---------|
| `UI.md` | Building Razor controls in the SDK |
| `Controls.md` | Quick checklist for creating and wiring controls |
| `Logic.md` | App class, actions, auth, and error flow |
| `Storage.md` | Storage selection, auth checks, and registration |
| `Entities.md` | Payload and entity patterns |
| `I18N.md` | Translation keys and runtime usage |
| `Guide.md` | Suggested workflow for adding a new feature |
| `Testing.md` | Unit testing with bUnit, FakeItEasy, and direct control instantiation |
| `SDK-Operations.md` | Console workflows: create, install, payloads, run, registration, self-hosting, links |

Start with `Guide.md` for a new feature, then open the topic-specific file you need.
